About Junkiree

Junkiree is a Free and Open Source loadshedding viewer for Google Chrome.
This app is currently at beta stage and further updates and bug fixes will be provided through the app store or through the project page. You can learn more about the app from junkiree’s website or Github page.

Help and Issues

Any feedback is greatly appreciated. If you stumble upon a problem with the app or need help regarding anything, drop me an email or a tweet.
You can also post issues, suggest changes or view the source from junkiree’s official Github page
